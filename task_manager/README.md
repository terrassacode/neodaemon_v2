# Task Manager

`task_manager` is a minimal executive index for project state.

It is not a dashboard, schema system, automation layer, or long-form documentation area.

Use it to answer quickly:

- what the project is;
- current health;
- current priority;
- what is being worked on;
- what is blocked;
- next action;
- evidence for the current state.

## Anti-Growth Rule

No añadir nuevos campos, schemas, dashboards, scripts ni más project files hasta que neodaemon.json haya sido útil en al menos una revisión real.
