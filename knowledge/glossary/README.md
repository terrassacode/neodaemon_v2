# Glossary

## Purpose

`knowledge/glossary/` is reserved for stable NeoDaemon vocabulary.

It may define names, terms, abbreviations, and operating concepts after an explicit decision.

## Policy

This folder is documental only.

It does not change runtime behavior.

It does not replace `MEMORY.md`.

## Single Source of Truth

Each glossary entry must point to the source that made the term authoritative.

If a term is still experimental, do not promote it here.

## Creation rule

Do not create glossary documents without an explicit decision.

No glossary content is migrated in this seed.
