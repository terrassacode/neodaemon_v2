# NeoDaemon Knowledge

## Purpose

`knowledge/` is the minimal documental knowledge space for NeoDaemon.

It is used to organize stable, curated project knowledge after an explicit decision.

## Policy

This folder is documental only.

It does not change runtime behavior.

It does not replace `MEMORY.md`.

`MEMORY.md` remains the continuity file for main/private session memory policy.

## Single Source of Truth

Each future document in `knowledge/` must have a clear Single Source of Truth.

Do not duplicate active contracts, runtime facts, or project state without naming the source.

## Creation rule

Do not create new knowledge documents without an explicit decision.

This seed creates structure only. No content migration is included.
