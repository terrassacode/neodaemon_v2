# Project Scope — OpenClaw Knowledge Wiki

## Nombre

openclaw-knowledge-wiki

## Ruta raíz autorizada

```text
/openclaw/workspace/main/projects/openclaw-knowledge-wiki/
```

## Objetivo

Crear una wiki local, manual y aislada para conocimiento operativo de OpenClaw.

## Rutas permitidas

```text
/openclaw/workspace/main/projects/openclaw-knowledge-wiki/
```

## Rutas prohibidas

- /openclaw/core
- /openclaw/workspace/main/scripts
- /openclaw/workspace/main/systemd
- /openclaw/workspace/main/rag_store
- /openclaw/workspace/main/memory
- /openclaw/workspace/main/.openclaw
- /openclaw/workspace/main/context_repo
- /openclaw/workspace/main/dashboard-v2
- /openclaw/workspace/main/logs
- /openclaw/workspace/main/backups
- /openclaw/workspace/main/briefings
- /openclaw/workspace/git_clean
- /openclaw/.env
- /home/openclaw/.openclaw

## Datos permitidos

- fuentes públicas o autorizadas colocadas manualmente en `raw/`;
- notas generadas dentro de `wiki/`;
- índice y log del propio proyecto;
- resultados de revisión manual dentro de `outputs/`;
- documentación local del proyecto.

## Datos prohibidos

- secretos;
- tokens;
- credenciales;
- `.env` reales;
- logs globales;
- memoria operativa privada;
- dumps de Gmail/Telegram;
- datos RAG internos;
- datos privados no autorizados.

## Reglas críticas

- `raw/` es inmutable;
- no modificar archivos dentro de `raw/`;
- `wiki/` es generado por Neodaemon;
- `wiki/index.md` es catálogo maestro;
- `wiki/log.md` es append-only;
- máximo 3 fuentes por ingest;
- no usar APIs externas;
- no instalar dependencias;
- no ejecutar scripts globales;
- no escribir fuera de la carpeta del proyecto.

## Reglas de ejecución

- no ejecutar scripts globales;
- no invocar `ru_event.sh`;
- no invocar `ru_interaction.sh`;
- no modificar servicios;
- no tocar gateway;
- no tocar systemd;
- no tocar RAG;
- no tocar Gmail;
- no tocar Telegram;
- no tocar `dashboard-v2`;
- no tocar logs globales;
- no tocar `memory`.

## Rollback

Eliminar o archivar únicamente:

```text
/openclaw/workspace/main/projects/openclaw-knowledge-wiki/
```

## Validación manual previa

Antes de escribir:

- comprobar rutas propuestas;
- comprobar que todos los archivos quedan dentro de `/openclaw/workspace/main/projects/openclaw-knowledge-wiki/`;
- comprobar que no se modifica `raw/`;
- comprobar que no se toca core, scripts globales, dashboard-v2, logs, memory, RAG, systemd, gateway, Gmail ni Telegram;
- comprobar que no se usan APIs externas;
- comprobar que no se instalan dependencias;
- comprobar que no se ejecutan wrappers globales.
